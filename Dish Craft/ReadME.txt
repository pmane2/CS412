
Dish Craft Project
This Dish Craft project is a web application developed using HTML, CSS, and PHP. The primary tools utilized for development are Visual Studio Code (VS Code) for code editing and XAMPP for local server deployment.

Features
User Interface and Landing Page:

The project includes an engaging user interface and a well-designed landing page to provide a user-friendly experience.

Database:

The application uses a MySQL database named dishdb.
The SQL script for creating the necessary tables and structure is provided in the dishdb.sql file.
How to Run the Project
Download and Install XAMPP:

Ensure that you have XAMPP installed on your machine. If not, download and install it from XAMPP.

Move Project Files:

Move the cloned project files to the htdocs directory of your XAMPP installation.
Start XAMPP:

Start the Apache server and MySQL in your XAMPP control panel.
Import Database:

Open phpMyAdmin in your browser (usually http://localhost/phpmyadmin).
Create a new database named dishdb.
Import the structure and data by selecting the dishdb.sql file.
Access the Application:

Open your web browser and navigate to http://localhost/dish-craft.
Use the provided admin credentials to log in and explore the application.
